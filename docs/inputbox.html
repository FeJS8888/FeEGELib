InputBox 输入框

文本输入框控件，支持光标、选择和 IME

返回首页

�� 特性

  • 光标定位和闪烁动画
  • 文本选择和拖拽
  • IME 输入法支持
  • 自动文本滚动
  • 波纹点击效果

主要方法

setContent(const std::wstring& s) - 设置输入内容
getContent() - 获取输入内容
setMaxlen(int maxlen) - 设置最大输入长度
setTextHeight(double height) - 设置文字高度

示例

InputBox* input = InputBoxBuilder()
    .setIdentifier(L"nameInput")
    .setCenter(400, 300)
    .setSize(300, 40)
    .setRadius(8)
    .setTextHeight(20)
    .setMaxLength(50)
    .build();

// 获取输入内容
std::wstring text = input->getContent();
