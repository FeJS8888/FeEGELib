Slider 滑动条

可拖动的滑动条控件

返回首页

特性

  • 水平和垂直方向支持
  • 步进值设置
  • 值改变回调
  • 平滑动画

主要方法

setProgress(double v) - 设置进度值 (0.0-1.0)
getProgress() - 获取当前进度
setStep(double s) - 设置步进值
setOrientation(Orientation ori) - 设置方向 (Row/Column)
setOnChange(callback) - 设置值改变回调

示例

Slider* slider = SliderBuilder()
    .setCenter(400, 300)
    .setSize(300, 20)
    .setProgress(0.5)
    .setStep(0.1)
    .setColor(EGERGB(200,200,200), EGERGB(100,180,255))
    .setOnChange([](double val) {
        std::cout << "进度: " << val << std::endl;
    })
    .build();
